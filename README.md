 Raigad Fort 360° View
📌 Project Overview

Raigad Fort 360° View is an interactive panoramic experience that allows users to explore Raigad Fort in a full 360-degree immersive view.
Users can move their mobile device or drag the screen to look around the fort environment, creating a virtual tour experience.

This project is designed to help students and history enthusiasts explore historical places digitally.

🎯 Purpose

The main goal of this project is to provide an immersive learning experience about Raigad Fort, the capital fort of Chhatrapati Shivaji Maharaj.

It helps:

Students learn history in an interactive way

Users experience forts virtually

Preserve and promote historical heritage using technology

✨ Features

360° panoramic view of Raigad Fort

Mobile Gyroscope support

Touch drag to rotate view

Smooth panorama rendering

Works inside Android WebView

Lightweight and fast loading

🚀 How to Run

Download or clone the project.

Open the project folder.

Run index.html in a browser.

For Android:

Place the files inside the assets folder.

Load the page using WebView.

📱 Controls
Action	Function
Drag Screen	Rotate 360° view
Move Phone	Gyroscope rotation
Pinch Zoom	Zoom in/out
🏰 About Raigad Fort

Raigad Fort is a historic hill fort located in Maharashtra, India.
It served as the capital of the Maratha Empire under Chhatrapati Shivaji Maharaj.

The fort is known for:

Maha Darwaja

Raj Sabha

Takmak Tok

Shivaji Maharaj Samadhi
